Elks Calendar Publisher

A themed monthly newsletter calendar, driven by Odoo Calendar.

Elks Calendar Publisher turns the lodge's standard Odoo Calendar (calendar.event) into the single source of truth for what's happening at the Lodge. From that one set of events, the module produces a printable monthly newsletter PDF and a live, embeddable calendar widget on the public website. No second calendar to maintain. No design tool to fight with each month.

Banners that own the day cell

Flag any event as a Banner — Queen of Hearts, Loudmouth Bingo, Bingo at the Lodge, Lodge Meeting, Live Music, Grace Bible Church, Special Event, Lodge Closed — and that event visually dominates its day cell on the published calendar. A higher-priority Banner wins the cell if two collide. Standard events render as small text lines below.

Editable graphic library

Each banner style ships with a matching graphic — a crowned heart for Queen of Hearts, an open mouth for Loudmouth Bingo, a treble clef for Live Music, a gavel for Lodge Meeting, and so on. Pick the banner style on an event and the matching graphic auto-fills. The library is editable, so the Lodge can add charity logos, event posters, or one-off uploads for a specific event without touching code. Every stock graphic is inline SVG, so it stays sharp on the printed calendar and on a phone.

Stock graphics ship with the module

A theme for every month

Thirteen stock themes are bundled — snowflakes for January, hearts for February, shamrocks for March, florals for April, blossoms for May, summer sun for June, the American flag for July, harvest for September, pumpkins for October, acorns for November, evergreen for December, plus an Elks-fraternal theme for special editions. Each theme is just a primary colour, a secondary colour, and an optional decorative SVG, so the Lodge can add custom themes from the Settings menu without writing code.

One source, two outputs

The Secretary picks a month, year, and theme, then clicks Generate PDF. The same QWeb template that drives the printed newsletter also renders the public-facing /elks/calendar page, and a JSON feed at /elks/calendar/json/<year>/<month> powers an embeddable Website Builder snippet that any editor can drop onto a page. One set of events, three places to see them.

Inherits Odoo Calendar — doesn't replace it

The module extends the built-in calendar.event model via _inherit. The standard Calendar UI keeps working exactly as Odoo ships it; the lodge gets four extra fields on the event form (Banner Style, Label, Sub-line, Priority) plus a Graphic picker. Recurring events like Loudmouth Bingo and the Lodge Meeting inherit the banner setting automatically on every recurrence. New fields are all namespaced elks_* so they won't collide with future Odoo upgrades.
